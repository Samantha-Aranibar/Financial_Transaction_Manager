# Flask Transaction Manager

A simple web application built with Flask that allows users to manage financial transactions through a browser.

The application demonstrates the implementation of CRUD (Create, Read, Update, Delete) operations using Python, Flask, and HTML templates.

---

## Features

- View all transactions
- Add a new transaction
- Edit existing transactions
- Delete transactions
- Responsive Bootstrap interface

---

## Technologies

- Python
- Flask
- HTML5
- Bootstrap
- Jinja2 Templates

## Installation

Clone the repository

```bash
git clone https://github.com/yourusername/Flask-Transaction-Manager.git
```

Create a virtual environment

```bash
python -m venv venv
```

Activate it

### Windows

```bash
venv\Scripts\activate
```

### macOS/Linux

```bash
source venv/bin/activate
```

Install dependencies

```bash
pip install -r requirements.txt
```

---

## Running the Application

```bash
python app.py
```

Then open

```
http://localhost:5000
```

## Current Limitations

Transactions are stored in memory and are reset whenever the application restarts.

Future versions could integrate:

- SQLite
- PostgreSQL
- MySQL

for permanent storage.
